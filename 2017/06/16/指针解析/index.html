指针解析

发表于 2017-06-16

1、指针的类型：指的是指针自身的类型
int *p;指针p的类型是int*
2、指针指向的类型：指的是指针指向的数据的类型
int *p指针p指向的类型是int
3、指针的值：指的是指针指向的数据的值，会被编译器当成一个地址。位置是从指针的值的代表的内存地址开始，长度为sizeof(指针指向的类型)
4、指针占据的内存大小=sizeof(指针的类型)
